This dialog is displayed whenever the user requests to open/save files from/to the disk. As seen in the following image, this window is divided into several areas:

"Look in" combo box: Select the logical disk where the files should be located.
"Current path" field: Displays the full path of the folder whose contents is currently displayed in the list.
"Add to favorites" button: This is used to add the current folder to the list of favorite folders. Each favorite folder is also displayed in the left sidebar (see below).
Left sidebar: Displays several special Windows folders such as:
  • Desktop
  • Favorites (Windows favorites, not application favorites)
  • CD Burning folder
  • My Documents
  • Recent files
It also lists all application defined favorite folders.
File list: Displays the files in the current folder that have the extension matching with one of the types selected in the "File type"combo box.
"File name" text box: This displays the name of the target file. The user can also type the name of the target file here.
"File type" combo box: This field defines the extensions of the files that are displayed in the list.
The contents of this combo box differs depending on the extension of the file(s) need to be opened/saved.

CAUTION: Not all the file dialogs have the feature of adding favorite folders; thus, not all file dialogs will display the "Add to favorites" button.
