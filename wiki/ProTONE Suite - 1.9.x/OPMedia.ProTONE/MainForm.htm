The main window of ProTONE Player is used to:

  • control the media playback functionality
  • interact with the playlist
  • interact with the bookmarks
    •
     
    This functionality can be accessed using buttons, context menus or command shortcuts. It is grouped into several areas depending on the applicability, as seen in this image:

    Playback functionality

    The main playback functionality is available by clicking the buttons in the playback panel, by using the corresponding entries in the context menu, or by using the appropriate command shortcut, as depicted in the following table. The context menu is available by right-clicking anywhere in the playback panel.

    Command Description Button Context menu Primary shortcut Alternate shortcut
    Play Starts playback Z Media Play (for multimedia keyboards)
    Pause Pauses playback X Space
    Stop Stops playback C Media Stop (for multimedia keyboards)
    Play Previous Plays previous item in playlist V Media Previous Track (for multimedia keyboards)
    Play Next Plays next item in playlist B Media Next Track (for multimedia keyboards)
    Load Files Loads one or more files N Select Media (for multimedia keyboards)
    Load DVD Loads a DVD medium Ctrl + D -
    Full Screen Toggles full screen mode Alt + Enter Ctrl + F
    Loop Play Toggles playing in loop Ctrl + L -
    Playlist End Event Enables notification at end of playlist Ctrl + E -
    Shuffle Play Toggles shuffle (random) play Ctrl + H -
    Settings Dialog Opens the settings window Ctrl + Alt + C -
       
    TIP:  Both the primary and alternate shortcuts can be changed using the Keyboard panel from Settings dialog
    TIP: The context menu can be also displayed by right-clicking the application icon in the system notification area:
     
    TIP: The kind of notification executed at the end of playlist is configurable via the Timer panel from Settings dialog and includes turning off the PC, enter in hibernate mode, or launching a custom program.

    Beside the above depicted functionality, related with playback are also the functions of media seek (that is - rewind and fast forward) and volume control. These are controlled with the time scale and volume scale, with the context menu, or with the corresponding keyboard shortcuts:

    Command Context menu Primary Shortcut Alternate shortcut
    Forward 5 seconds Right Arrow Period (. key)
    Rewind 5 seconds Left Arrow Comma (, key)
    Volume up by 5% Add (+ key) Volume Up (for multimedia keyboards and most of laptops)
    Volume down by 5% Subtract (- key) Volume Down (for multimedia keyboards and most of laptops)
       
    TIP:  Both the primary and alternate shortcuts can be changed using the Keyboard panel from Settings dialog
    TIP:  The Context menu also contains functional replicas of the time and volume scales. These can be used as well for media seeking and for controlling the playback volume.
     

    Playlist functionality

    The playlist functionality is available by clicking the buttons in the playlist panel, by using the corresponding entries in the context menu, or by using the appropriate command shortcut, as depicted in the following table. The context menu is available by right-clicking anywhere in the playlist panel.

    Command Description Button Context menu Primary shortcut Alternate shortcut
    Move Up Moves selected items one position up in playlist Page Up  Key 9 from numeric pad 
    Move Down Moves selected items one position down in playlist Page Down  Key 3 from numeric pad 
    Delete Removes the selected items from playlist Delete   Decimal (. from numeric pad)
    Clear Clears the playlist (removes all items) Ctrl + Delete  Ctrl +   Decimal (. from numeric pad)  
    Load Loads a saved playlist.
    Supported formats: PLS, ASX, WPL or M3U.    		 Ctrl + N  Ctrl + Select Media (for multimedia keyboards)
    Save Saves the current playlist in one of the supported
    playlist formats (ASX, PLS, WPL or M3U).    		 Ctrl + S -

    TIP:  The playlist buttons toolbar can be displayed or hidden by using the Playlist extender button. The application will always start with this toolbar hidden (regardless of the state it had when the previous application session has been closed).
    TIP:  When we right-click an playlist item that has bookmarks defined, an additional entry will be added to the playlist context menu. This entry has as children all defined bookmarks. The user can choose to which of these bookmarks to jump with the playback.
    TIP: When we right click a DVD playlist item, an additional entry will be added to the playlist context menu. This entry has as children all titles, chapters, and languages that are available within the DVD. The user can choose to which of these titles of chapters to jump with the playback, or he can choose the desired subtitle language.
    TIP: For playlist items representing video media files, an additional entry will be added to the playlist context menu. By means of this menu entry, the user has the option to download a subtitle for the video file from the Internet. The options related to subtitle download are defined in the Subtitle panel from Settings dialog.
    CAUTION: Currently, it is not possible to download subtitles for DVD's. This feature is not yet implemented, so the embedded DVD subtitles will be used instead. The options related to DVD subtitles are defined in the Subtitle panel from Settings dialog.
    CAUTION: It's not possible to define bookmarks for DVD items (mostly because the DVD media are not writable). Instead, the existing titles and chaptes act like bookmarks. The Bookmark Manager will not open when a DVD item is selected.

    Bookmarks functionality

    The bookmarks functionality is available by bringing up the Bookmark Manager panel, by using the Bookmark Extender button or by using the corresponding command shortcut (Ctrl + Alt + B).

    With the Bookmark Manager it is possible to add new bookmarks or to edit the text or the time for the already defined bookmarks. These bookmarks are saved in a file having the same location and name as the media file but with the extra .BMK extension. Basically, bookmarks act like some stamps indicating some particular spots within the media file. Typically an user will define a bookmark at the place when the media file contains an important or interesting passage for him/her.

    To define a new bookmark, use the "Add" button.

    		This button is always visible.
    To define a new bookmark for the current playback time, use the "Add now" button. This button is visible only when the currently played file is open in Bookmark Manager.
    After using this button, the playback will be paused, giving the user some time to enter the bookmark description.

    To delete an existing bookmark, use the "Delete" button.

    		This button is visible only when a bookmark is selected.
       
    TIP: To edit the time or the description of a bookmark, simply click the corresponding cell in the list.
    TIP: The application will always start with the Bookmark Manager hidden (regardless of the state it had when the previous application session has been closed).
