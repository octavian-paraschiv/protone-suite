From the main window of ProTONE Player, the user is able to:

  • Control the media playback e.g. play/pause/stop etc.
  • Interact with the playlist e.g. load/save playlist or arrange playlist items etc.
  • View and alter the media properties e.g. view sample frequency or alter the ID3 tags for MP3 files etc.
  • View signal diagrams like VU-Meter, Waveform and Spectrogram
  • Interact with the bookmarks list e.g. define new bookmarks, adjust bookmark times or delete bookmarks.
    •

    Control media playback

    You can control the media playback using:

  • The playback area
    • groups all the interface elements that appear at the bottom of the main application window.
    These elements appear always, regardless which is the selected page: Playlist / Track Info / Signal Analisys / Bookmarks.
    The following image depicts which is the area named "playback area":

    1. The first row in the playback area is occupied by the playback buttons. These buttons can be used to control the playback e.g. play/pause/stop etc. They all have tooltips and suggestive icons so their function is self explanatory. Still on the first row, but far in the right, are the state icons. These icons show the current playback status, the availability of audio/video streams, and the media type icon. Hovering with the mouse over these icons will reveal explanatory tooltips.

    2. The second row is occupied by the time scale. Clicking anywhere on the time scale will result in playback jumping immediately to that particular moment (instant rewind/fast forward). Hovering with the mouse over the time scale will reveal a tooltip depicting the corresponding playback time.
    If the hovered position on the time scale is in a 5-seconds range to a defined bookmark, the tooltip will contain the bookmark description also.

    3. Finally, the third (and last) row in the playback area is occupied by the volume scale. Clicking anywhere on the volume scale will result in volume changing immediately to that particular level. Hovering with the mouse over the volume scale will reveal a tooltip depicting the corresponding volume level.

  • The playback context menu
    • that is available by right-clicking anywhere in the playback area.

    From the context menu you can access the same set of functionalities that are available via the Playback area.
    Moreover, some functionalities, like 'Open URL', are available ONLY from the context menu.
    This context menu is also shown when right clicking on the application icon in the system notification area (area that is shown near the clock, usually in the bottom right of the screen, known also as 'system tray').
