From the main window of ProTONE Player, the user is able to:

  • Control the media playback e.g. play/pause/stop etc.
  • Interact with the playlist e.g. load/save playlist or arrange playlist items etc.
  • View and alter the media properties e.g. view sample frequency or alter the ID3 tags for MP3 files etc.
  • View signal diagrams like VU-Meter, Waveform and Spectrogram
  • Interact with the bookmarks list e.g. define new bookmarks, adjust bookmark times or delete bookmarks.
    •

    Control media playback

    You can control the media playback using:

    A. The playback area groups all the interface elements that appear at the bottom of the main application window. These elements appear always, regardless which is the selected page (Playlist, Track Info, Signal Analisys or Bookmarks). The following image depicts which is the area named "playback area":

    1. The first row in the playback area is occupied by the playback buttons. These buttons can be used to control the playback e.g. play/pause/stop etc. They all have tooltips and suggestive icons so their function is self explanatory. Still on the first row, but far in the right, are the state icons. These icons show the current playback status, the availability of audio/video streams, and the media type icon. Hovering with the mouse over these icons will reveal explanatory tooltips.

    2. The second row is occupied by the time scale. Clicking anywhere on the time scale will result in playback jumping immediately to that particular moment (instant rewind/fast forward). Hovering with the mouse over the time scale will reveal a tooltip depicting the corresponding playback time.
    If the hovered position on the time scale is in a 5-seconds range to a defined bookmark, the tooltip will contain the bookmark description also.

    3. Finally, the third (and last) row in the playback area is occupied by the volume scale. Clicking anywhere on the volume scale will result in volume changing immediately to that particular level. Hovering with the mouse over the volume scale will reveal a tooltip depicting the corresponding volume level.

    B. The playback context menu that is available by right-clicking anywhere in the playback area.

    From the context menu you can access the same set of functionalities that are available via the Playback area.
    Moreover, some functionalities, like 'Open URL', are available ONLY from the context menu.
    TIP: This context menu is also shown when right clicking on the application icon in the system notification area (area that is shown near the clock, usually in the bottom right of the screen, known also as 'system tray').

    View and alter the media properties

    Use the Track Info page to view and alter the media properties, such as the Duration, Title or Artist.

    To view and edit media properties for a specific file, select it from the list on the left side. Then edit the properties using the grid on the right side. NOTE:
    1. This list is a "lightweight" replica of the playlist. Any changes done to the playlist are reflected here too.
    2. Depending on the selected file, it might not be possible to edit its properties. For example, information for CD tracks or for video files is not editable. Also it is not possbile to view or edit the info for streamed media like Shoutcast radio stations.
    The editable information is always displayed under the "Media Tag" property group.
    Currently only the MP1/MP2/MP3 files have editable properties.
    But, even for such a file, the properties within may still not be editable in some cases (see Note #2 above).

    View the signal diagrams

    Use the Signal Analisys page to view signal graphics such as VU-Meter, Waveform and Spectrogram.

    Note that signal analisys requires significant processing power (CPU speed), so, if your computer is not very powerful (slow CPU), you can disable this feature or parts of it.
    This can be done by means of the Misc -> Main Pages tab from the Settings dialog

    Interact with the bookmarks list

    Use the Bookmarks page to add new bookmarks or to edit the text or the time for the already defined bookmarks. These bookmarks are saved in a file having the same location and name as the media file but with the extra .BMK extension. Basically, bookmarks act like some stamps indicating some particular spots within the media file. Typically an user will define a bookmark at the place when the media file contains an important or interesting passage for him/her.

    To view and edit bookmarks for a specific file, select it from the list on the left side. NOTE:
    1. This list is a "lightweight" replica of the playlist. Any changes done to the playlist are reflected here too.
    2. Depending on the selected file, it might not be possible to edit its bookmarks. For example, if the file is on a Read-Only disk (e.g. a CD-ROM), then the bookmarks are visible but not editable. Also it is not possbile to view or edit bookmarks for streamed media like Shoutcast radio stations.

    To define a new bookmark, use the "Add" button.

    		This button is always visible.
    To define a new bookmark for the current playback time, use the "Add now" button. This button is visible only when the currently played file is open in Bookmark Manager.
    After using this button, the playback will be paused, giving the user some time to enter the bookmark description.

    To delete an existing bookmark, use the "Delete" button.

    		This button is visible only when a bookmark is selected.
    To edit the time or the description of a bookmark, simply click the corresponding cell in the list.
